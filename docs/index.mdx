# @manuscripts/manuscript-editor

This module exports several components for editing, viewing, navigating and manipulating a manuscript: 

* `Editor` is a manuscript editor.
* `Viewer` is a manuscript viewer.
* `DebouncedManuscriptOutlineContainer` is an outline view of a manuscript.
* `ManuscriptToolbar` is a renderer for toolbar buttons that act on a manuscript.
* `ApplicationMenu` is a renderer for menu items that act on a manuscript.
